Modules

What are Modules?

Modules are like JavaScript libraries that include a set of functions we can include in our application.

Node.js has a set of built-in modules that we can use without any further installation.

To include a module, we can use the require() function with the name of the module:


    var http = require('http');

Now our application has access to the HTTP module, and is available for server creation:


    http.createServer(function (req, res) {
        res.writeHead(200, {'Content-Type': 'text/html'});
        res.end('Hello World!');
    }).listen(8080);

Creating Modules

We can create our own modules and require them in our applications. Following is an example of a module that returns the current date and time:


    exports.myDateTime = function() {
      return Date();
    }

The exports keyword makes properties and methods available outside the module file.

Create a file called date-time.js and insert the following code:


    exports.myDateTime = function() {
      return Date();
    };

Require the date-time modules in our app.js file using the following:


    // Require Date-Time Module
    var dt = require('./modules/date-time.js');

Now we can print out the result of the date-time module in our browser using the following in our app.js file:


    http.createServer(function(req, res) {
      res.writeHead(200, {'Content-Type': 'text/html'});
      res.write("The current date and time are: " + dt.myDateTime());
    }).listen(8080);
